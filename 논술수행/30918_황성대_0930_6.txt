데이터 전처리
데이터 전처리(data preprocessing)는 데이터 분석을 하기 위해 수집 데이터 분석을 하기 위해 수집 데이터를 가공하여 분석에
적합한 데이터 형태로 변환하는 모든 작업을 의미한다. 데이터 전처리 대상에는 주로 데이터 결측값과 데이터 이상값이 있다.
데이터 전처리 방법에는 데이터 정제, 데이터 변환, 데이터 축소, 이상 데이터 처리 등 다양한 작업 단계가 존재한다.
데이터 분석가 업무 시간의 70~80%가 데이터 전처리 작업에 사용되며, 데이터 전처리 결과에 따라 데이터 분석의 정확도가
높아지기 때문에 데이터 전처리는 매우 중요한 작업 단계이다.

1. 데이터 전처리 유형
데이터 전처리는 데이터 정제와 분석 변수 처리로 나뉜다.

구분
데이터 정제

분석 변수 처리

유형
데이터 실수화 (data vectorization)

데이터 정제 (data cleaning)

데이터 통합 (data integration)

데이터 축소 (data reduction)

파일 변수 생성 (dervied variance)

데이터 변환/변수변환 (data transformation)

불균형 데이터 처리 (data balancing)

설명
데이터 실수화는 문자, 범주 등의 데이터를 실수형으로 변환하는 작업을 의미
(사례) 지도 학습에는 실수형 데이터만 사용 가능하므로 문자 변수를 그에 맞는 실수형 변수로 바꾸는 데이터 실수화를 수행


가장 많은 사용되는 데이터 전처리 작업유형 중 하나로서.
데이터 결측값, 이상값, 부정확한 레코드를 탐지하고 제거하거나 수정하여 데이터 분석할 때 잘못된 분석 결과가 나오는 것을 사전에
방지하는 데이터 전처리 방식

서로 다른 데이터베이스, 데이터 세트, 혹은 파일을 하나로 합치고 정리하는 과정
(사례) 원화, 달러와 같은 서로 다른 화폐단위를 사용하는 두 개의 파일을 동일한 화폐 단위로 변환 후 통합

데이터 분석을 효과적이고 효율적으로 하기 위해 다수의 변수들중, 중요한 변수들을 선정하는 직업. 변수 선택과 차원 축소로 구성
[변수 선택(variable selection)]: 데이터 분석을 위해 고려되는 다양한 다수의 변수들 중에서 독립변수를 선택하는 과정.
독릭변수는 종속변수에 영향을 주는 결정적인 요소이므로 올바른 독립변수 선택은 매우 중요한 데이터 변수 처리 과정
[차원 축소(dimensionality reduction)]: 데이터 모델링의 복잡도를 줄이고 분석 시간을 단축하기 위해 차원 축소가 필요. 
종속변수에 영향을 주지 않는 선에서 독립변수의 수를 줄이는 작업

변수끼리 조합하거나 함수를 적용하여 새로운 변수를 생성하는 데이터 전처리 유형
(사례) 반 학생들 성적을 평균 내어 성적 평균이라는 신규 변수를 생성하고 성적 평균 칼럼에 저장

데이터의 형식이나 구조에 변경을 가한 후, 다른 형식이나 구조로 변경하는 데이터 전처리 유형
데이터 스케일링(data scaling): 데이터 표준화, 정규화, 분포와 범위 조정 등을 통해 데이터를 변환하는 작업

데이터 분석을 위해 수집한 데이터 세트 간, 범위나 숫자의 차이가 너무 크거나 하는 이유로 편향된 분석 결과가 발생하는 것을 방지하기
위해 데이터 샘플링을 통해 데이터 세트 간 비중을 유사하게 조정하는 작업
